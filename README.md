# Symfony api template
this project is meant to serve as a template/boilerplate project for basic api stuff, using symfony,doctrine and apiplatform in order to do so

## pre requisite :
- PHP 8.5+
- Composer
- Docker and Docker Compose
- Symfony CLI

## intended usage :
api creation is done via the creation of entity and the use of [API Platform](https://api-platform.com/docs/symfony/)
